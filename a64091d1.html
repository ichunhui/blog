Kong

Kong 是一个云原生、快速、可扩展和分布式的微服务抽象层（也称为 API 网关，API 中间件）。

使用 Kong

  • 启动（必须确保执行过 kong migrations up） - kong start [-c /path/to/kong.conf]
    • -c /path/to/kong.conf 参数用来指定用户的配置
  • 停止 - kong stop
  • 重启 - kong reload
